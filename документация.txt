================================Как настроить проект?================================

что бы настроить проект должен быть обязательно файл settings.fs
в этом файле должно быть строчки

облегченный код.fl
с++ код.cpp

название файлов может быть любое главное что бы были указаны и созданы

после создания документов можете открыть программу и запустить и если программа не выдаст ошибки то вы все настроили правильно

что бы проверить как пишется код откройте ваш облегченный код и впишите в него следующий код на мигание светодиода

========
установка
	пин мод 7 выход
все

код
	подать 7 вкл
	задержка 100
	подать 7 выкл
	задержка 100
все
========

запустите этот код и в файле c++ должен появится следующий код

========
void setup(){
    pinMode( 7, INPUT);
}
void loop(){
    digitalWrite( 7, true );
    delay( 100 );
    digitalWrite( 7, false );
    delay( 100 );
}

========

если все так то все хорошо


================================Обзор базового кода================================

ВСЕ ЗНАЧЕНИЯ ДЛЯ ФУНКЦИЙ ДАВАТЬ ЧЕРЕЗ ПРОБЕЛ ОБЯЗАТЕЛЬНО

установка - эта функция записывает в основной код void setup() функция которая выолняется в начале программы
в ней можно настроить пины и т.д.

пин мод - функция для установки пина записывает в основной код pinMode(pin, value);
pin - номер пина для установки
value - принимает вход и выход. Записывает в основной код INPUT или OUTPUT вход или выход для пина

все - функция которая заканчивает функцию или условие. В нашем случае она заканчивает функцию "установка"

код - функция которая записывает основной код void loop()
между ней записывается главный код для обработки чего либо. Работает в бесконечном цикле

подать - функция которая подает ток на пин записывает в основной код digitalWrite(pin, value);
pin - номер пина для подачи
value - значение вкл или выкл подает ток или нет
+ перегрузка
если написать подать силу то функция принимает номер пина и силу подаваемой на пин
записывает в основной код analogWrite(pin, value);
pin - номер пина
value - подаваемая сила от 0 до 255 

задержка - функция задержки программы записывает в основной код delay(value);
value - время задержки в милисекундах




================================Переменные================================

переменная - принимает тип переменной число или флаг и название переменной к примеру
переменная число а
записывает в основной код int а;

так же переменной сразу можно указать значение к примеру 
переменная число а = 3
записывает в основной код int а = 3;

так же есть тип переменных флаг (bool в с++)
у нее есть только 2 значение вкл и выкл (true или false в с++)
так же как и числу можно сразу задать значение


задать - функция дающая переменной какое либо значение к примеру
задать а 3

так же можно дать значение входа пина к примеру
задать а вход 7
7 - это номер пина с которого берется значение
выдает 2 значения вкл или выкл

можно взять силу которая идет с пина к примеру
задать а вход силы 7
7 - номер пина с которого берется вход
выдает число  


можно так же воспользоватся заменой которая будет заменять одно на другое к примеру
заменять а 3
если в коде будет написано а, то в компиляции оно будет заменятся на 3

================================Условия================================

если - функция проверяющая условие принимает условие записывает в основной код if ( condition )
condition - условие которое выполняется

можно проверить значение переменной операторами

== ровно
> больше
< меньше
>= больше или ровно
<= меньше или ровно
!= не ровно

записывается к примеру 

если а==5
    код который будет только если переменная а будет ровна 5
все

так же есть иначе записывает else в код с++ 
пример кода с иначе

если а==5
    код который будет только если переменная а будет ровна 5
все
иначе
    код который будет если основное условие будет не верно
все
